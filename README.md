# 🥾 Hiking Safety Tracker

A personal safety app built for solo hikers.

## Features
- Real-time weather conditions for any trail location
- Hike logging with distance, duration, and notes
- Automated safety check-in system — sends emergency email alert if you don't check in by your expected return time
- Built with Python, Flask, SQLite, OpenWeatherMap API
